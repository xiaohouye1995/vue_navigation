# vue3-admin-liuguang

**中文** | [English](./README.md)

## 构建步骤

```bash
# 安装依赖(建议用pnpm)
# 你可以运行npm i -g pnpm 去安装pnpm
pnpm

# 启动服务
pnpm run dev
```

浏览器访问 http://localhost:5001


## 发布

```bash
# 构建测试环境
pnpm run build-test

# 构建生产环境
pnpm run build
```

## 其它

```bash
# 预览生产环境
pnpm run preview

# 代码格式检查并自动修复
pnpm run lint
```
